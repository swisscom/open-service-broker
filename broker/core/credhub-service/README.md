# CredHub Service

A service for using [CredHub](https://docs.cloudfoundry.org/credhub/)  
 
## Usage
In your Spring boot application `build.gradle`, include the following dependency:
```$groovy
dependencies {
    compile 'com.swisscom.cloud.sb:credhub-service:6.7.0'
```

## Build
Run:
```bash
$ ./gradlew build
```
